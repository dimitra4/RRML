## Regex



**Attributes:**

| Name | Type | Status | Description | Constraints |
|:-----|:-----|:-------|:------------|:------------|
| `column` | str | Optional | The name of the column to which the regular expression will be applied. It must match one of the defined relation keys |  |
| `pattern` | str | Optional | The regular expression pattern |  |
| `groups` | List[int] | Optional | The grouping where the pattern will be applied |  |
